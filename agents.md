Основные файлы проекта

Ассоциация Героев (Мир One-Punch Man).pdf — основной файл приключения. Он содержит полную запись игровой сессии, которая служит базой для последующего анализа и тестирования технологий.

log.md - в данный файл выводится каждое действие, выполненное агентом. Каждый агент, перед выполнением PR, должен записать в этот файл следующий последовательный номер действия, дату, а также подробное описание работы, которая была выполнена, а также промт, по которому можно воспроизвести выполненную работу на другом приключении на этом же этапе.

rules.md — дополняемый файл, в котором находятся основные правила, механики и детали сеттинга (например, классификация угроз, ранговая система, учет разрушений). ИИ-мастер должен руководствоваться этими правилами при проведении приключения.



Что такое приключение (Adventure)

В контексте данного проекта «приключение» — это интерактивная нарративная сессия, построенная по принципам настольных ролевых игр (D\&D):



Ролевая составляющая: ИИ берет на себя роль Мастера (DM), описывающего мир и реакции окружения, а игрок управляет конкретным персонажем в рамках заданного сеттинга.



Механика заявок: Игрок описывает намерения своего героя (что он хочет сделать или сказать).



Фактор случайности: Исход каждого значимого действия определяется броском кубика. Результат броска напрямую влияет на то, как ИИ интерпретирует развитие событий (успех, провал или критическое последствие).



Нарративный цикл: Приключение представляет собой последовательность итераций «Заявка — Бросок — Описание результата», формирующую целостный сюжет.



Как был получен этот файл

Файл является результатом экспорта и обработки игровой сессии в сервисе StoryForge:



Настройка: ИИ-генератору были заданы параметры сеттинга и детальное описание игрового персонажа.



Игровой процесс: Прохождение сессии в реальном времени, где каждое действие фиксировалось в текстовом логе вместе со значениями выпавших кубиков.



Трансформация: По завершении приключения накопленные данные (тексты и метаданные) были выгружены и сконвертированы в PDF-формат для удобного хранения и последующего анализа через другие ИИ-инструменты.
