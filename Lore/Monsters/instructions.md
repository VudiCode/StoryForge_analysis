# Инструкция по созданию сущности категории "Монстры" (Monsters)

Каждый монстр или группа монстров (угроза) имеет свой собственный каталог, название которого соответствует имени монстра (на английском или транслите без пробелов).
В каталоге создается файл `LoreObject.md`, который содержит актуальное состояние монстра на текущий момент.
Если в главе произошли изменения в состоянии монстра, создается файл `<Номер главы>.md` с состоянием на момент окончания главы (например, `1.md`).

## Жесткая структура файла LoreObject.md (и файлов истории <Номер главы>.md):

```markdown
# [Имя Монстра / Название угрозы]

**Уровень угрозы:** [Волк, Тигр, Демон, Дракон, Бог]
**Текущий статус:** [Жив, Мертв, Неизвестно]
**Местоположение:** [Город, район]

## Описание
[Внешний вид, происхождение, способности и особенности поведения монстра.]

## Разрушения
[Описание причиненного ущерба.]
```
