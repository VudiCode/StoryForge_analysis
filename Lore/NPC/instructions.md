# Инструкция по созданию сущности категории "НПС" (NPC)

Каждый НПС имеет свой собственный каталог, название которого соответствует имени НПС (на английском или транслите без пробелов).
В каталоге создается файл `LoreObject.md`, который содержит актуальное состояние НПС на текущий момент.
Если в главе произошли изменения в состоянии НПС, создается файл `<Номер главы>.md` с состоянием на момент окончания главы (например, `1.md`).

## Жесткая структура файла LoreObject.md (и файлов истории <Номер главы>.md):

```markdown
# [Имя НПС]

**Роль:** [Роль в сюжете, профессия]
**Класс/Ранг:** [Если является героем: S/A/B/C и ранг, иначе "Отсутствует"]

## Описание
[Описание НПС, внешний вид, характер, отношение к главному герою и другим фракциям.]

## Связи
- [Связанный персонаж/фракция] - [Описание связи]
```
