# Экзоскелет Мк-1

**Тип:** Броня/Механизм
**Владелец:** Кайто Сато

## Состояние
Незавершён, ~40% боеготовности. Левая рука функциональна на 60%, правая отсутствует. Из нагрудной пластины торчат неизолированные провода. Питание нестабильно. Пригоден только для ограниченного использования.

## Описание и свойства
Самодельный экзоскелет, собираемый Кайто Сато из обломков дронов и прочего технологического мусора на задворках складского квартала Города Z. Представляет собой каркас с амбициями стать боевой броней.
