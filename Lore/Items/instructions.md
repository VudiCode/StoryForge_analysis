# Инструкция по созданию сущности категории "Предметы" (Items)

Каждый предмет имеет свой собственный каталог, название которого соответствует названию предмета (на английском или транслите без пробелов).
В каталоге создается файл `LoreObject.md`, который содержит актуальное состояние предмета на текущий момент.
Если в главе предмет был изменен, поврежден или улучшен, создается файл `<Номер главы>.md` с состоянием на момент окончания главы (например, `1.md`).

## Жесткая структура файла LoreObject.md (и файлов истории <Номер главы>.md):

```markdown
# [Название предмета]

**Тип:** [Оружие, броня, инструмент, ресурс и т.д.]
**Владелец:** [Имя персонажа/НПС/Фракции или "Нет"]

## Состояние
[Текущее состояние предмета, процент целостности, уровень заряда и т.д.]

## Описание и свойства
[Подробное описание предмета, его происхождение и уникальные свойства/характеристики.]
```
