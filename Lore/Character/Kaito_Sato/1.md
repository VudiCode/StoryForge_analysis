# Кайто Сато

**Прозвище:** ожидает утверждения
**Класс:** C
**Ранг:** 341

## Статус
**Здоровье:** 100% — Физически в норме, хотя ночь без сна за паяльником даёт о себе знать лёгкой резью в глазах и тремором в пальцах.
**Энергия / Спец. ресурс:** 80% — Голова ясная, но кофе был давно. Экзоскелет Мк-1 собран лишь частично: левая рука на 60%, правая отсутствует, питание нестабильно. Пригоден только для ограниченного использования.

## Репутация
**Значение:** 8% — Неизвестен широкой публике. В базе Ассоциации числится как «новичок без подтверждённых достижений». Местные жители складского квартала видели его пару раз, но считают чудаковатым бездомным.

## Инвентарь / Экипировка
- Экзоскелет «Мк-1» (незавершён, ~40% боеготовности — только левая рука)
- Рабочий ноутбук (с хакерскими скриптами и ПО для перехвата управления техникой)
- Самодельный электрошокер ×2 (заряд на 3–4 разряда каждый)
- Моток промышленного кабеля (15 м)
- Базовый набор инструментов (паяльник, мультитул, изолента)

## Описание
Начинающий герой C-класса, проживающий в бывшем грузовом контейнере на задворках складского квартала Города Z. Занимается сборкой собственного экзоскелета из мусора и запчастей, которые остаются после битв других героев. Стремится помогать людям, несмотря на бюрократию Ассоциации Героев.
