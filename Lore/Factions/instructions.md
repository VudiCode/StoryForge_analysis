# Инструкция по созданию сущности категории "Фракции" (Factions)

Каждая фракция имеет свой собственный каталог, название которого соответствует названию фракции (на английском или транслите без пробелов).
В каталоге создается файл `LoreObject.md`, который содержит актуальное состояние фракции на текущий момент.
Если в главе произошли изменения во фракции, создается файл `<Номер главы>.md` с состоянием на момент окончания главы (например, `1.md`).

## Жесткая структура файла LoreObject.md (и файлов истории <Номер главы>.md):

```markdown
# [Название фракции]

## Описание
[Общее описание фракции, ее цели, идеология, методы работы.]

## Влияние и статус
[Текущее влияние в мире, отношения с другими фракциями или героями.]

## Известные представители
- [Представитель 1]
- [Представитель 2]
```
